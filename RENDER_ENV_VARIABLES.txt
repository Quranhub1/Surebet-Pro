==============================================
RENDER DEPLOYMENT NOTES & ENVIRONMENT VARIABLES
==============================================

⚠️ Build failed because Render is trying to deploy this repo as a *Static Site* and is looking for a publish directory called "root".

✅ This project is a **Node.js Web Service** (Express server) and should be deployed as a **Web Service**, not a Static Site.

### ✅ Render Service Setup (Web Service)
1. In Render Dashboard, create a **New Web Service** (not Static Site).
2. Connect your GitHub repo and select this project.
3. Set these values:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`

> If you already created a Static Site, remove it and create a Web Service instead.

---
